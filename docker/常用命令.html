# docker run

  • --name 启动后的容器名
  • -d 后台发布
  • -p 端口映射 本机:容器
  • -e 环境变量
  • -v 挂载项 本机路径:容器路径
  • mysql 新建后会根据 /docker-entrypoint-initdb.d 里的数据文件如*.sql逐一运行初始化数据库
  • 本地如果有安装mysql并且为了不占用3306设置关闭状态, mysql -uroot -pxxxx命令不能访问容器

# docker exec

https://docs.docker.com/engine/reference/commandline/exec/

  • -i``--interactive 交互式
  • -t``--tty 可以理解是容器内部的命令行暴露到本地
