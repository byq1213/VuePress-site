    # 运算符相关面试题

    # 以下代码输出什么

    console.log(1+"2")
console.log(1-"2")
    解析 
    console.log(1+"2") => "12"
console.log(1-"2") => -1

    + 具有有连接字符串的功能，从左到右执行，遇到字符串就连接字符串。 - 只具有数学运算减法的功能，字符串"2"转换为数字2
