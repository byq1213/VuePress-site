# 自我介绍

山西人 17 年大专毕业， 在太原做了 1 年半小程序全栈工作。学到了很多东西但由于太原的软件环境不理想，没有进步空间也没有发挥价值，遂 在 2019 年来到深圳，在腾讯理财通前端组做外包驻场工作。 主要负责维护组内内网系统，及一些h5 c 端页面的开发。 日常会和后台对接一些数据接口。配合组内对某些工具系统做一些优化改造（图片上传、测试账号申请、日志查询）等工作 主要技术栈是 vue+ nodejs，部分系统也用到了 react，

前端这边也会自己去配置自动化部署 CI,达到快速开发。
